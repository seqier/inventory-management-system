# 木马牛进销存系统

#### 介绍
基于TP5开发的进销存系统 

#### 软件架构
基于以下技术开发

1. BUI
2. Bootstrap3
3. jquery
4. Auth权限控制
5. AutoComplete
6. cxselect联动菜单
7. php-Excel库
8. C-Lodop网页打印组件
9. thinkphp5.0.24
10. datetimepicker


实现了如下功能：

1. 生产管理
2. 入库、出库
4. 库存查询、调拨、报废
5. 财务管理
6. 系统用户权限访问控制
7. 仓库、供应商、会员管理


#### 安装教程

1. 下载代码到本地目录
2. 如果放在域名下运行，请将网站目录指向到public目录
3. 新建mysql数据库，并导入demo_ims.sql文件
4. 修改application目录下的database.php文件，修改为你当前的数据库参数
5. 安装完成
